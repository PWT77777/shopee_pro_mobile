# ✨ 蝦皮好物分享 (Shopee Pro Mobile v3.6)

一款專為蝦皮分潤斜槓者打造的**精品級行動端海報產生器**。透過智慧解析與美學排版，讓你 30 秒產出極具吸引力的推坑圖，大幅提升轉單率！

---

## 🌟 核心特色

- 🤖 **智慧解析引擎 v3.1**：自動從蝦皮分享文字中精準提取商品名稱、價格與分潤連結。
- 🎨 **精品設計模板**：內建 VOGUE 雜誌、黑金奢華、文青拍立得、科技霓虹等 6 款專業風格。
- 📐 **動態堆疊佈局 (Flow Layout)**：標題再長也永不重疊，智慧適應 9:16 限動或 1:1 方圖比例。
- 💎 **頂級字體支援**：內建 Noto Serif TC、Outfit、Playfair Display 等頂級美學字體。
- 🌈 **自定義美學**：自由調整主題色調與標題字型，打造個人品牌專屬感。
- 📱 **行動裝置優化**：手機瀏覽器完美適配，隨時隨地即時產圖。

---

## 🚀 快速上手

1. **複製資訊**：在蝦皮 App 點擊分享並複製商品連結文字。
2. **智慧填充**：貼入工具中的「智慧解析區」，點擊自動偵測。
3. **上傳美照**：選擇一張商品的乾淨去背圖或美妝圖。
4. **選定風格**：切換喜歡的模板、顏色與字體。
5. **長按儲存**：在手機上長按生成的畫布即可儲存圖片，並分享至 Threads、IG 或 FB。
